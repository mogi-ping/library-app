# Library App

Консольное приложение для учёта книг в библиотеке.

## Описание

Приложение позволяет:
- добавлять книги
- просматривать каталог
- выдавать книги читателям
- возвращать книги
- смотреть список выданных книг

## Технологии

- Python 3
- SQLite (sqlite3)
